# CS 340 Portfolio Reflection

Throughout this course, I learned the importance of writing programs that are maintainable, readable, and adaptable. One of the best examples of this was the CRUD Python module I created in Project One. By separating the database operations into a reusable module, I was able to connect the dashboard widgets in Project Two without rewriting database code. This approach made the application easier to organize, troubleshoot, and update. If changes were needed in the database connection or queries, they could be made in one location rather than throughout the entire dashboard application. In the future, this CRUD module could be reused in other projects that require MongoDB database access, saving development time and promoting code reuse.

When approaching the requirements provided by Grazioso Salvare, I used a structured problem-solving process. I first analyzed the client's needs, identified the data that would be required, and then developed queries to retrieve the correct information from the database. After validating the queries, I integrated them into the dashboard to create interactive visualizations and filtering options. This project differed from many previous assignments because it required connecting multiple components, including the database, Python module, dashboard interface, and data visualizations. In future projects, I would continue using a modular design approach, carefully gathering requirements before implementation and creating flexible database structures that can adapt to changing client needs.

Computer scientists develop solutions that help organizations collect, manage, analyze, and use information more effectively. Their work matters because businesses rely on accurate data and efficient systems to make informed decisions. Through this project, I created a dashboard that allows Grazioso Salvare to quickly identify dogs that meet specific rescue training requirements without manually reviewing thousands of records. The database and dashboard improve efficiency, reduce the potential for human error, and provide decision-makers with useful information in a format that is easy to understand. Projects like this demonstrate how computer science can help organizations operate more effectively and achieve their goals through technology.
